# Variables and Types in Python

## What are Variables
Variables are containers used to store data values in memory. In Python, you create a variable by assigning a value to a name using the "=" operator. Python is dynamically typed, meaning you do not need to declare the type of a variable explicitly.

## Basic Data Types

### Integer
Integers are whole numbers, positive or negative, without decimals.

Example:

x = 10

y = -5

### Float
Floats are numbers that contain decimal points.

Example:

price = 19.99

temperature = -3.5

### String
Strings are sequences of characters enclosed in quotes (single or double).

Example:

name = "Santiago"

message = 'Hello World'

### Boolean
Booleans represent one of two values: True or False.

Example:

is_active = True

is_logged_in = False

## Key Concepts
- Variables are created when you assign a value
- Python does not require explicit type declarations
- Variable names should be descriptive and follow naming conventions
- You can reassign variables to different types
- Use type() to check the data type of a variable

## Summary
Variables store data, and Python provides basic data types like integers, floats, strings, and booleans to represent different kinds of information.
